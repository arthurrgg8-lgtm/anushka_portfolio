# Bhotey Kukur — Himalayan Mastiff Portal

A premium, high-performance web portal for Nepal's premier Himalayan Mastiff (Bhotey Kukur) breeding kennel. Built with a modern full-stack architecture to showcase champion bloodlines with cinematic visuals and mobile-first design.

## 🚀 Tech Stack

- **Frontend:** Next.js 16 (App Router), React 19, Tailwind CSS 4, TypeScript.
- **Backend:** FastAPI (Python), Uvicorn, Pydantic.
- **Data:** Centralized JSON-driven content management.
- **Visuals:** Framer Motion-inspired animations, Glassmorphism UI, Responsive TikTok integration.

## ✨ Key Features

- **Champion Showcase:** Cinematic, parallax-driven gallery of top breeding dogs.
- **Mobile-First Design:** Fully responsive UI with specialized mobile navigation and touch-optimized layouts.
- **Dynamic Pricing:** Multi-tier puppy reservation system with eSewa integration.
- **Video Gallery:** Compact, high-density TikTok grid view for kennel updates.
- **Interactive Contact:** Direct WhatsApp and Email integration with embedded location maps.

## 🛠️ Setup & Installation

### Backend
1. Navigate to `backend/`
2. Create virtual environment: `python -m venv venv`
3. Activate venv: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run server: `python main.py` (Starts on port 8000)

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev` (Starts on port 3000)
